## Installation

1. Install NPM packages.

```sh
npm install
```

2. Create .env file in the root folder.

```sh
touch .env
```

3. Copy and paste your firebase config and user information into the .env file

```env
FIREBASE_API_KEY=
AUTH_DOMAIN=
PROJECT_ID=
STORAGE_BUCKET=
MESSAGEING_SENDER_ID=
APP_ID=

USER_EMAIL=
USER_PASSWORD=
```
